
=== FTDI USB SERIAL OVERVIEW ===

This driver enables the EfiSerialIoProtocol interface
for FTDI based USB-to-Serial adapters.

=== STATUS ===

Current status: Alpha

Serial Input: not-working (disabled by default)
Serial Output: functional

=== COMPATIBILITY ===

Tested with:
OVMF using QEMU parameters: -usb -usbdevice serial::vc
